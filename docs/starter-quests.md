# Starter Quests

These are the first 10 quests for the hackathon MVP. Each quest should create one useful inventory item or one visible project update.

## Quest Design Rule

Every quest must answer three questions:

- What real problem does this solve?
- What input does the player provide?
- What reusable asset enters the inventory?
- What project, room, skill book, or workflow does it update?

## 1. Forge A Raw Idea

Dwarf: Orin Forgekeeper

Player input:

- Rough idea
- Target user
- Desired output
- Related project, if any

Agent task:

- Clarify the idea.
- Turn it into a stronger prompt.
- Add output format and constraints.

Reward item:

- Prompt Scroll
- Optional Project Map update

Completion condition:

- Player saves the refined prompt and optionally links it to a project.

## 2. Build A 7-Day Sprint

Dwarf: Chrono Deepclock

Player input:

- Project
- Goal
- Available time
- Deadline
- Constraints

Agent task:

- Break the goal into a 7-day execution plan.
- Identify the riskiest dependency.
- Define daily deliverables.

Reward item:

- Sprint Compass
- Project milestone update

Completion condition:

- Player accepts the plan or edits it once.

## 3. Price The First Offer

Dwarf: Ledger Goldhand

Player input:

- Product or service idea
- Target customer
- Expected value
- Delivery cost

Agent task:

- Suggest 3 pricing models.
- Compare pros and cons.
- Recommend the first pricing experiment.

Reward item:

- Pricing Rune

Completion condition:

- Player chooses one pricing model.

## 4. Map The Market Gate

Dwarf: Mira Marketseer

Player input:

- Product idea
- Target audience
- Current distribution channels

Agent task:

- Define the likely ICP.
- Suggest 3 acquisition channels.
- Write a simple positioning statement.

Reward item:

- Market Map

Completion condition:

- Player saves one positioning statement.

## 5. Automate A Repetitive Task

Dwarf: Tessa Toolwright

Player input:

- Manual task
- Current tools
- Trigger
- Desired output

Agent task:

- Convert the task into an automation workflow.
- Suggest tools or APIs.
- Identify failure points.

Reward item:

- Automation Gear
- Workflow Blueprint

Completion condition:

- Player saves the workflow blueprint and can place it in a project room or workflow forge.

## 6. Break The Weak Assumption

Dwarf: Brakka Battlemason

Player input:

- Business or product idea
- Why the player believes it will work

Agent task:

- Identify the top 5 assumptions.
- Rank them by risk.
- Design the smallest validation test.

Reward item:

- Assumption Hammer

Completion condition:

- Player chooses one assumption to test first.

## 7. Capture A Hackathon Lesson

Dwarf: Liora Lorekeeper

Player input:

- Hackathon project
- What happened
- What was learned
- What would be done differently

Agent task:

- Turn the lesson into a structured story.
- Extract reusable principles.
- Suggest whether it belongs in a future book, post, or playbook.

Reward item:

- Lore Fragment

Completion condition:

- Player saves the story fragment.

## 8. Turn Prompt Into Workflow

Dwarf: Orin Forgekeeper

Player input:

- Existing prompt
- Repeated use case
- Desired repeatable result

Agent task:

- Convert the prompt into a reusable workflow.
- Define inputs, steps, checks, and output format.

Reward item:

- Workflow Blueprint
- Workflow Machine

Completion condition:

- Player saves the workflow and places it into the workflow forge or a project room.

## 9. Create A Founder One-Liner

Dwarf: Mira Marketseer

Player input:

- Product
- Audience
- Pain point
- Outcome

Agent task:

- Generate 5 one-liners.
- Score them for clarity and specificity.
- Recommend the strongest one.

Reward item:

- Offer Scroll

Completion condition:

- Player picks one one-liner.

## 10. Open The First Project Room

Dwarf: Orin Forgekeeper

Player input:

- Project name
- Project type
- Current goal
- Existing inventory item, if any
- Next action

Agent task:

- Create a simple project profile.
- Recommend which dwarf should help next.
- Attach one useful item or quest to the room.
- Define the room's visible status.

Reward item:

- Project Map

Completion condition:

- Player creates the first project room and sees at least one linked item, quest, or next action.

## MVP Quest Mix

The first 10 quests cover:

- Prompting
- Execution
- Monetization
- Positioning
- Automation
- Strategy
- Storytelling
- Knowledge management
- Project visualization

This is enough to prove the game loop without building combat, multiplayer, or a huge map.
