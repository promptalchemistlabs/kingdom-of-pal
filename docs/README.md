# Kingdom of PAL Docs

Kingdom of PAL is an open-world spatial interface for Prompt Alchemist Labs where players walk through a 2.5D/3D kingdom to talk to dwarven AI agents, manage workflows, collect skill books, and visualize real projects.

The product is not a Minecraft clone. Minecraft is only a loose inspiration for open exploration and player agency. The closer interaction model is a RuneScape-like world: click or keyboard movement, NPC conversations, inventory, quests, skill progression, and repeatable activities.

OpenClaw is the agent runtime and execution layer. Kingdom of PAL is the embodied world layer that makes those agents, projects, workflows, and knowledge assets explorable.

## Current Product Stance

Build the smallest open world that makes AI agents, projects, workflows, and knowledge feel embodied, useful, and memorable.

The first version should prove:

- Players want to talk to agent NPCs inside a world.
- Inventory and quests make AI outputs feel earned, reusable, and visible.
- Projects can be represented as physical places, boards, rooms, or machines.
- Dwarven agents can represent Prompt Alchemist Labs' knowledge, services, and community.
- The world can become a distribution surface for workshops, templates, tools, and paid products.

## Working Principle

Do not start by building a massive world.

Start with one polished district, seven memorable dwarfs, ten useful quests, one project room, and an inventory system that turns agent outputs into reusable assets.
